# Ariana & Juan 2026 — Transparent Web Brand Assets Pack

This pack recreates the web brand assets from the original uploaded monogram image, with the black background removed and replaced by transparency.

## Included files

### Favicons and browser icons
- favicon.ico
- icons/favicon-16x16.png
- icons/favicon-32x32.png
- icons/favicon-48x48.png

### Apple / Android / PWA icons
- icons/apple-touch-icon.png
- icons/android-chrome-192x192.png
- icons/android-chrome-512x512.png
- icons/maskable-icon-192x192.png
- icons/maskable-icon-512x512.png
- site.webmanifest

### Social sharing images (transparent PNG)
- social/og-image.png
- social/twitter-card.png
- social/whatsapp-preview.png
- social/social-square.png

### Setup helpers
- head-tags.html
- source/ (reference and transparent master source files)

## Notes
- Background has been removed from the recreated assets and is transparent wherever the format supports alpha.
- Maskable icons were exported with the monogram centered inside the inner safe zone for Android launchers.
- Replace https://YOUR-DOMAIN.com in head-tags.html with your real production URL.
